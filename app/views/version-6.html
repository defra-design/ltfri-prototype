Version 6.0

Insights from user research

Users still struggling with the amount of content on the maps, in order to reduce cognitive load we are testing whether only showing them what they need to know and is relevant to them at first will help in understanding their risk. The option in the toggle allows curious users or technical users to enable more features within the map.

Users finding it difficult to locate the scenario control as it is fixed to the left hand side, so moving it the right, where the key is should help in keeping interactions and focus in one area. This is especially pronounced in wide screen users, where the scenario control is very far from the key and it takes users a while to notice it is even there.

Mobile user testing revealed that it was difficult for users to figure out which icon would open up the key, so adding text to the button should provide enough context for them to navigate the map.
