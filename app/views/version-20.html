Version 20 shows the latest content and designs for Groundwater and Reservoir content, for the risk overview page and 'more about..' pages.

V20 changes

  • Groundwater content has been added in V20 and has been tested. We needed to add the reservoir content (that won't be tested) to sit alongside the groundwater content.

  • Contents pages have also been updated to match the live service more closely, for exmaple, the content list to navigate to other pages, was situated in the navigation, this doesn't need to be placed here, as this rule only generally applies if there are links to a a new action, as per GDS guidelines, so it has been replaced by the standard GOV UK contents list.

  • A design decision for how we show the results of the Groundwater and Reservoir risk outcome has been decided and is shown in this prototype

A note on binary risk options

  • For groundwater and reservoir, they are both binary outcomes, but we will only be showing the result that is relevant. All content for each of the binary outcomes can be found on the copy deck.
